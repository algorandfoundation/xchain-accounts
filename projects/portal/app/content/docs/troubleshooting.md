# Troubleshooting

Solutions to issues you may run into while using xChain EVM.

## Mobile

### Transactions aren't showing up in my wallet

If you tap **Review** on mobile and the signing prompt never appears in your wallet, the connection between the dApp and the wallet is likely being dropped by the mobile OS or browser when the app switches to the background.

**Try the wallet's built-in browser instead.** Most major mobile wallets ship with an integrated dApp browser. Using that browser keeps the dApp and wallet in the same app, which gives a much more reliable connection - signing prompts arrive immediately and don't get lost when switching contexts.
